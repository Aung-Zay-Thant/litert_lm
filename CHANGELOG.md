## 0.1.0

- Initial release
- On-device LLM inference with LiteRT-LM engine
- Streaming text generation with `generateStream()`
- Multimodal support (text + image prompts)
- Cancel generation mid-stream
- System prompt configuration
- Conversation reset
- Model download from Hugging Face with resume support
- Secure token storage
- iOS support via native C API bridge
- Android support via official LiteRT-LM Kotlin SDK
